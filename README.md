# Currency Converter

A lightweight and responsive currency conversion web application built using vanilla JavaScript. The application fetches live exchange rates from an external API and dynamically updates conversion results in real time.

## Features
- Real-time currency conversion
- Supports 150+ currencies
- Dynamic country flags
- Network error handling
- Clean and minimal UI

## Technologies Used
HTML, CSS, JavaScript (ES6), Fetch API

## Deployment
Hosted using GitHub Pages.

## Future Enhancements
- Swap currency button
- Auto conversion on input change
- UI improvements
- Dark mode support
